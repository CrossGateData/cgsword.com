生產品收集任務
　
職業限定 資深生產系 備註 無
必要等級 Lv 推薦等級 Lv
時間限制 無 起始地點 亞諾曼城的旅館2樓
重複完成 可 頭目戰 無
任務獎賞 傳說(4級品收集)、技藝(5級品收集)、神人(6級品收集)

1.首先到亞諾曼城的旅館(151，151)的二樓找四級品搜集者(15，17)說話，他會要你收集的物品清單如下：
武器類：雙刃長劍　　或　鎖刃劍
　　　　長戰斧　　　或　雙刃斧
　　　　穗槍　　　　或　雙叉戟
　　　　狙擊之弓　　或　超強狙擊弓
　　　　賢者令牌　　或　賢者手杖

　　　　銳角回力標　或　獅子回力標
　　　　騎士小刀　　或　耳柄小刀

防具類：霸葛頭盔　　或　角飾頭盔
　　　　時髦帽子　　或　鴨舌帽
　　　　鋼鐵鎧甲　　或　金屬皮甲
　　　　布甲　　　　或　襯甲
　　　　烏鴉之袍　　或　月亮之袍
　　　　鎖鏈長靴　　或　鎖鏈靴
　　　　薄板鞋　　　或　鎖鏈鞋
　　　　士兵盾　　　或　重盾

補給品類：燒雞　　　或　親子丼 (1個)

　　　　　＋250的藥水 (1瓶)

2.武器及防具必須要是龜裂的，全部收集完成後交給四級品搜集者(15，17)，可以換得『傳說』。

3.接著向五級品特搜員(16，18)說話，接下收集任務，他會要你收集的物品清單如下：

武器類：格鬥劍　　　　或　長鍔劍
　　　　鉤爪斧　　　　或　闊刃斧
　　　　護衛槍　　　　或　騎士槍
　　　　殺手長弓　　　或　超衝擊弓
　　　　琥珀杖　　　　或　琥珀令牌

　　　　翼鹿回力標　　或　阿修塔羅回力鏢
　　　　球鍔小刀　　　或　雙刃小刀

防具類：開放式頭盔　　或　葛理克頭頭盔
　　　　藍徽章之帽　　或　高級帽子
　　　　長形金屬鎧甲　或　索狀鎧甲
　　　　環服　　　　　或　士兵護衣
　　　　東方法衣　　　或　聖堂之袍
　　　　蜥蝪靴　　　　或　蜥蝪長靴
　　　　蜥蜴鞋　　　　或　維京鞋
　　　　蜥蝪盾　　　　或　反光之盾

補給品類：漢堡　　　　或　星鰻飯團 (1個)

+300藥水(1瓶)

4.武器及防具必須要是龜裂的，全部收集完成後交給五級品特搜員(16，18)，可以換得『技藝』。

5.接著向六級品搜集迷(10，16)說話，接下收集任務，他會要你收集的物品清單如下：

武器類：豬牙劍　　　　　或　雙刃闊劍
　　　　彎月闊刃斧　　　或　大型闊刃斧
　　　　大砍槍　　　　　或　長尖槍
　　　　巨弓　　　　　　或　長格鬥弓
　　　　元素之杖　　　　或　神聖之杖
　　　　蛇獅回力標　　　或　鶇回力鏢
　　　　闊刃小刀　　　　或　觸角小刀

防具類：巴比克頭盔　　　或　利刃頭盔
　　　　裝飾著花的帽子　或　青銅帽子
　　　　重鐵板甲　　　　或　鐵板半身甲
　　　　長索錐衣　　　　或　索錐上衣
　　　　聖者之袍　　　　或　戰鬥長袍
　　　　鋼靴　　　　　　或　鋼製長靴
　　　　戰鬥鞋　　　　　或　騎士鞋
　　　　風之盾　　　　　或　獅子之盾

補給品類：炒麵麵包　　　或　壽喜鍋 (1個)
