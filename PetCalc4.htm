一級與野生寵物能力範圍預測公式與實作
(1)純白/兔耳/嚇人箱各等級理論最低血量公式
嚇人每級最低可能血量 = (等級+4) * 9.16 + 20
純白每級最低可能血量 = (等級+4) * 8.76 + 20
兔耳每級最低可能血量 = (等級+4) * 8.48 + 20
(2)公式解析
中間這個參數是用該種寵的各項底段檔次* (8 2 3 3 1) * 0.04 去算的

0.04等同除以25，是上一篇提過的野寵成長係數公式，因為現在是在計算下限，所以用最低的成長係數來計算。

上面公式的+4是由寵物1級每檔的 0.2 / 0.04 = (1 + 4)得來的，35級就是用35 + 4 = 39

檔次計算時把隨機檔全配在魔上面(加魔BP得到的血最少)

例如說純白檔次是(9 24 29 11 37+10)，全底檔次(每項BP成長各掉4檔)是(5 20 25 7 43)

　(5 20 25 7 43) * (8 2 3 3 1) / 25 = (5 * 8 + 20 * 2 + 25 * 3 + 7 * 3 + 43 * 1) / 25
= 219 / 25
= 8.76

龍城半路開箱遇到一隻70級純白的話可以算出最低血量是：
(70+4) * 8.76 + 20 = 668.24

要封印抓起來的話，打到接近這個數字時就可以不用再打了，用同樣的原理可以算出各級野生寵的所有能力下限。
(3)野寵上限能力計算
要算上限也是可以的，不過這時要代入最高的成長係數5

以算純白每級的最高血量為例：
一級的基本每檔0.2除上成長係數公式的0.045 = 40/9 = 1 + 3.44444...

檔次計算時把隨機檔全配在體上面(加體BP得到的血最多)

純白全頂檔次是(9 + 10 24 29 11 37)

　(19 24 29 11 37) * (8 2 3 3 1) * 0.045 = (19 * 8 + 24 * 2 + 29 * 3 + 11 * 3 + 37 * 1) 0.045
= 357 * 0.045
= 16.065

純白每級最高可能血量 = (等級-1 + 40 / 9) * 16.065 + 20

70級純白最高可能血量 = (69 + 40 / 9) * 16.065 + 20 = 1199.9

所以70級野生純白血量介乎於 668 ~ 1199

用同樣的原理可以算出所有一級 / 野生寵的能力範圍。
